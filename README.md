# Base64Decode

[English](README.en.md) | [简体中文](README.zh-CN.md)

## Overview

Base64Decode is a lightweight Chrome extension for decoding selected Base64 text from the context menu.

## 概览

Base64Decode 是一个轻量级 Chrome 扩展，用于通过右键菜单解码选中的 Base64 文本。

## Bilingual Support

- Extension UI supports English (`en`) and Simplified Chinese (`zh_CN`) via Chrome i18n.
- Browser language decides which locale is shown.
- Feature introductions are available in both docs:
  - [README.en.md](README.en.md)
  - [README.zh-CN.md](README.zh-CN.md)

## Quick Features

- Decode Base64 from right-click selection.
- Show decoded result in-page near selected text.
- Copy decoded text with one click.
- UTF-8 decode with `gb18030` / `gbk` / `big5` fallback.

## 快速功能

- 右键解码选中的 Base64 文本。
- 在选区附近浮层展示解码结果。
- 一键复制解码内容。
- UTF-8 解码失败时回退 `gb18030` / `gbk` / `big5`。
